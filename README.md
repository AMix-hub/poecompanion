# PoE Companion

A comprehensive Path of Exile companion — available as both a **desktop application** (Windows `.exe`, macOS `.dmg`, Linux `.AppImage`) and a web app.

## Features

- **📖 Leveling Guide** — Act-by-act walkthrough for all 10 acts with zones, skill gems to collect, boss strategies, and expert tips
- **⚔ Items Database** — Browse weapons, armour, accessories, gems, flasks and currency with filtering by category, rarity, and full-text search
- **🔨 Crafting Simulator** — Interactive simulator using all major currency orbs (Alchemy, Chaos, Exalted, Annulment, etc.) with crafting bench support
- **🛡 Pro Builds** — Curated builds sourced from top players (maxroll.gg, poe.ninja, community), filterable by difficulty, ascendancy and playstyle
- **📰 News** — Patch notes, league announcements, events, community highlights, and development updates with expandable full content

## Desktop Application (.exe / .dmg / .AppImage)

### Prerequisites

- [Node.js](https://nodejs.org) v18 or later
- On Windows: no extra tools needed
- On macOS: Xcode Command Line Tools (`xcode-select --install`)
- On Linux: `dpkg`, `fakeroot` for `.deb` targets (optional)

### Build the Windows installer (.exe)

```bash
npm install
npm run dist:win
```

Output is written to `dist-electron/`. You will find:

| File | Description |
|------|-------------|
| `PoE Companion Setup x.x.x.exe` | NSIS installer — installs the app and creates a desktop/start-menu shortcut |
| `PoE Companion x.x.x.exe` | Portable executable — run without installing |

### Build for other platforms

```bash
npm run dist:mac    # macOS .dmg
npm run dist:linux  # Linux .AppImage
npm run dist        # all three platforms at once
```

### Preview the desktop app without packaging

```bash
npm run build           # export Next.js to static files
npm run electron:preview  # opens the Electron window directly
```

## Web Application (browser)

### Development server

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Production web build

```bash
npm run build   # outputs static HTML/CSS/JS to out/
npm run start   # serve via Next.js (not needed for Electron)
```

## Project Structure

```
src/
  app/                # Next.js App Router pages
    page.tsx          # Home / Dashboard
    leveling-guide/   # Leveling Guide page
    items/            # Items Database page
    crafting/         # Crafting Simulator page
    builds/           # Pro Builds page
    news/             # News page
    __tests__/        # Unit tests
  components/         # Shared React components
    Navbar.tsx
    Footer.tsx
    ItemCard.tsx
    BuildCard.tsx
    NewsCard.tsx
  data/               # Static game data
    items.ts
    builds.ts
    news.ts
    leveling.ts
    crafting.ts
  types/              # TypeScript type definitions
    index.ts
electron/
  main.js             # Electron main process (loads out/ via electron-serve)
out/                  # Static export (generated by `npm run build`)
dist-electron/        # Desktop installers (generated by `npm run dist:win` etc.)
```
